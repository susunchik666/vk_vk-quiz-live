# VK Quiz Live

VK Quiz Live — MVP веб-приложения для проведения интерактивных квизов в реальном времени. Проект разработан в рамках онлайн-практики VK.

## Описание проекта

Приложение позволяет организатору создавать квиз, запускать комнату по уникальному коду, а участникам — подключаться к активному квизу и отвечать на вопросы в режиме реального времени.

После завершения квиза система подсчитывает баллы участников и отображает итоговый лидерборд.

## Основные функции

* создание квиза организатором;
* подключение участников по коду комнаты;
* отображение вопросов в реальном времени;
* отправка ответов участниками;
* подсчёт баллов;
* показ итоговой таблицы результатов.

## Технологии

* HTML
* CSS
* JavaScript
* Node.js
* Express
* Socket.IO

## Запуск проекта

Установить зависимости:

```bash
npm install
```

Запустить сервер:

```bash
npm start
```

После запуска открыть в браузере:

```text
http://localhost:3000
```

## Цель проекта

Цель проекта — разработать работоспособный прототип платформы для проведения интерактивных квизов на мероприятиях, где взаимодействие между организатором и участниками происходит в реальном времени.
