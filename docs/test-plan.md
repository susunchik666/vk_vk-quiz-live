# План тестирования MVP

## Сценарий 1. Регистрация

1. Создать пользователя-организатора.
2. Создать пользователя-участника.
3. Проверить вход по email и паролю.

## Сценарий 2. Создание квиза

1. Войти как организатор.
2. Создать квиз.
3. Добавить минимум два вопроса.
4. Проверить, что квиз появился в личном кабинете.

## Сценарий 3. Запуск комнаты

1. Нажать «Запустить комнату».
2. Проверить, что появился код комнаты.
3. Открыть вторую вкладку и войти как участник.
4. Подключиться по коду комнаты.

## Сценарий 4. Реальное время

1. Организатор запускает первый вопрос.
2. Участник видит вопрос без обновления страницы.
3. Участник отправляет ответ.
4. Организатор видит обновление лидерборда.

## Сценарий 5. Завершение квиза

1. Организатор завершает квиз.
2. Все пользователи видят итоговый лидерборд.
3. Результат сохраняется в истории.
