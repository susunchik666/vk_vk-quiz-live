# Макеты экранов для Figma

Эти экраны можно быстро перенести в Figma и получить ссылку на макет для сдачи проекта.

## Экран 1. Авторизация

- Название: VK Quiz Live
- Подзаголовок: Интерактивные квизы в реальном времени
- Форма: email, пароль, кнопка входа, ссылка на регистрацию
- Переключатель роли при регистрации: организатор / участник

## Экран 2. Личный кабинет организатора

- Приветствие пользователя
- Кнопка: создать новый квиз
- Список созданных квизов
- Кнопка: запустить комнату

## Экран 3. Создание квиза

- Поля: название, категория, время на вопрос, правила
- Блоки вопросов
- Тип вопроса: один ответ / несколько ответов
- Ссылка на изображение
- Варианты ответов
- Отметка правильного ответа

## Экран 4. Комната организатора

- Код комнаты крупно по центру
- Список участников
- Кнопки: запустить первый вопрос, следующий вопрос, завершить
- Предпросмотр текущего вопроса

## Экран 5. Экран участника

- Ввод кода комнаты
- Текущий вопрос
- Варианты ответа
- Кнопка отправки ответа
- Сообщение о результате ответа

## Экран 6. Лидерборд

- Рейтинг участников
- Баллы
- Место участника
- Финальный статус квиза
